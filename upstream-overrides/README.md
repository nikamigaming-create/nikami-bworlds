# Complete changed upstream files

This directory contains complete final source files for the small set of
upstream xNVSE/xOBSE files modified by Nikami. Copy them over the same relative
paths in the pinned upstream checkout after adding the direct oracle source from
`../oracles`.

The patch queues under `../patches` remain authoritative for code review and
exact tree replay. These complete files exist so the resulting C++ can be read
without mentally reconstructing a diff.
